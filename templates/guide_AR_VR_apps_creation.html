Практичний посібник з створення складних та динамічних AR/VR-застосунків у Unity

Практичний, детальний та покроковий алгоритм, щоб навіть початківець міг створити складний інтерактивний додаток (наприклад, AR-додаток для розміщення меблів з жестами, голосовим керуванням та динамічними анімаціями, або VR-сцену з взаємодією об'єктів).

Я фокусуюся на Unity як основному рушії, оскільки він популярніший для AR/VR-початківців (підтримує AR Foundation для кросплатформенності, має велику спільноту та легший інтерфейс). Як альтернатива, наприкінці розділу декілька слів про Unreal Engine.

Ілюстрація VR-окулярів: Символізує вступ до AR/VR-розробки в Unity.

Вимоги до вашого ПК/Мак:

  • ОС: Windows 10/11, macOS 10.15+ або Linux.
  • Процесор: Intel Core i5 або еквівалент, 8 GB RAM (рекомендовано 16 GB для складних проектів).
  • Диск: Мінімум 20 GB вільного місця (для Unity + SDK + інструментів).
  • Інтернет: Стабільне з'єднання (завантаження може зайняти години).
  • Пристрої для тестування: Android-смартфон з ARCore (наприклад, Google Pixel), iPhone з ARKit (iPhone 8+), або VR-гарнітура як Meta Quest 3.

Всі інструменти безкоштовні для особистих/освітніх проектів (платні версії для комерції). Вказую посилання, розміри, час установки (орієнтовний, залежить від швидкості інтернету 50+ Mbps) та точні кроки з меню.

1. Підготовка середовища розробки

Крок 1.1: Встановлення Unity

Unity — кросплатформенний рушій для AR/VR. Підтримує AR Foundation (для ARKit/ARCore), Vuforia, MRTK тощо.

  • Посилання: https://unity.com/download
  • Платність: Безкоштовно (Unity Personal для проектів з бюджетом < $100k/рік).
  • Розмір: Unity Hub ~100 MB (завантаження), Unity Editor з модулями ~2-3 GB (встановлений ~5-10 GB з AR/VR-модулями).
  • Час установки: 10-30 хв (завантаження Hub: 1-2 хв; Editor: 5-20 хв залежно від модулів).

Покрокова установка:

  1. Завантажте Unity Hub з сайту (виберіть вашу ОС).
  2. Запустіть інсталятор. У вікні "Unity Hub Setup" оберіть шлях (наприклад, C:\Program Files\Unity Hub), натисніть "Next" > "Install".
  3. Відкрийте Unity Hub. Натисніть "Sign in" (створіть акаунт, якщо немає).
  4. У Hub натисніть "Installs" > "Install Editor".
  5. Оберіть останню LTS-версію (наприклад, Unity 6000.0 LTS на 2025 рік). У вікні "Add modules" оберіть:
    • Android Build Support (для ARCore/Android).
    • iOS Build Support (для ARKit/iOS, потребує Mac).
    • Windows Build Support (якщо на Windows).
    • Додаткові: OpenXR (для VR), WebGL (якщо потрібно).
  6. Натисніть "Install". Чекайте завершення (прогрес у Hub).
  7. Створіть новий проект: "Projects" > "New Project" > Оберіть шаблон "3D" або "AR/VR" (якщо доступний), вкажіть ім'я (наприклад, "MyARApp"), натисніть "Create Project".
Ілюстрація Unity Editor: Показує інтерфейс для налаштування AR/VR-проектів.

Крок 1.2: Встановлення Android SDK (для Android/ARCore)

Потрібно для білду на Android-пристрої.

  • Посилання: https://developer.android.com/studio
  • Платність: Безкоштовно.
  • Розмір: Android Studio ~1 GB (завантаження), SDK з компонентами ~5-10 GB (встановлений до 52 GB, якщо всі API, але можна обмежити).
  • Час установки: 30-60 хв (завантаження: 5-10 хв; установка + SDK: 20-50 хв).

Покрокова установка:

  1. Завантажте Android Studio (виберіть .exe для Windows або .dmg для Mac).
  2. Запустіть інсталятор. У "Android Studio Setup" оберіть "Standard" > "Next".
  3. У процесі установки оберіть SDK-компоненти: Android SDK Platform (API 33+ для ARCore), Android SDK Tools.
  4. Після установки відкрийте Android Studio. Натисніть "SDK Manager" (іконка шестерні > SDK Manager).
  5. У SDK Manager оберіть вкладку "SDK Platforms" > Встановіть API 34 (Android 14) та 33.
  6. Вкладка "SDK Tools" > Встановіть "Android SDK Build-Tools", "Android Emulator" (якщо потрібно емулятор).
  7. Натисніть "Apply" > "OK". Чекайте завантаження (~2-5 GB).
  8. У Unity: Edit > Project Settings > Player > Android > Other Settings > Minimum API Level = 24 (для ARCore).
Ілюстрація Android Studio: Для налаштування ARCore на Android пристроях.

Крок 1.3: Встановлення iOS SDK (Xcode, для iOS/ARKit)

Тільки на Mac.

  • Посилання: https://developer.apple.com/xcode/ (або App Store).
  • Платність: Безкоштовно (потрібен Apple ID).
  • Розмір: Завантаження ~12 GB, встановлений ~23 GB (для Xcode 16+).
  • Час установки: 1-3 години (завантаження: 30-60 хв; розпакування: 30-120 хв, особливо на Intel Mac).

Покрокова установка:

  1. Відкрийте Mac App Store, пошукайте "Xcode", натисніть "Get" (або завантажте з developer.apple.com).
  2. Чекайте завантаження (прогрес в App Store).
  3. Після завантаження відкрийте Xcode. У першому запуску воно розпакує компоненти (може зайняти 30+ хв).
  4. У Xcode: Xcode > Settings > Platforms > Встановіть iOS SDK (якщо не автоматично).
  5. У Unity: Edit > Project Settings > Player > iOS > Other Settings > Target SDK = iPhone Only, Architecture = ARM64.
Ілюстрація Xcode: Для розробки ARKit на iOS.

Крок 1.4: Встановлення AR/VR SDK та плагінів у Unity

  • AR Foundation (кросплатформенний для ARKit/ARCore):
    • Безкоштовно, ~100 MB.
    • У Unity: Window > Package Manager > Search "AR Foundation" > Install (версія 6.0+).
    • Додайте провайдери: Search "ARCore XR Plugin" та "ARKit XR Plugin" > Install.
  • Vuforia (для маркер-based AR):
    • Безкоштовно базова версія (watermark), повна — платна.
    • Посилання: https://developer.vuforia.com/downloads/sdk
    • У Unity: Window > Package Manager > Add package from git URL > Вставте "com.ptc.vuforia.engine" > Add.
    • Або завантажте з Asset Store: Assets > Asset Store > Search "Vuforia" > Import.
  • MRTK (Mixed Reality Toolkit, для VR/AR з жестами):
    • Безкоштовно.
    • Посилання: https://learn.microsoft.com/en-us/windows/mixed-reality/mrtk-unity/
    • Завантажте Mixed Reality Feature Tool з https://aka.ms/MRFeatureTool (~50 MB, установка 5 хв).
    • У інструменті: Import MRTK3 packages > Select features (Core, UX, Input) > Import to Unity.
    • У Unity: Mixed Reality > MRTK3 > Configure > Assign Default Profile.
  • ARCore SDK (для Android):
    • Безкоштовно, ~50 MB.
    • Посилання: https://developers.google.com/ar/develop/downloads
    • Інтегрується через ARCore XR Plugin (вже в AR Foundation).
Ілюстрація SDK плагінів: Показує інтеграцію AR Foundation, Vuforia тощо в Unity.

Крок 1.5: Налаштування емуляторів та пристроїв

  • Android емулятор: У Android Studio > AVD Manager > Create Virtual Device > Оберіть Pixel з ARCore > System Image API 30+.
  • Meta Quest (для VR):
    • Посилання: https://developers.meta.com/horizon/documentation/native/android/mobile-device-setup/
    • Безкоштовно.
    • У мобільному app Meta Quest: Menu > Devices > Ваш Quest > Headset Settings > Developer Mode > Enable.
    • Під'єднайте USB: У Unity Edit > Project Settings > XR Plug-in Management > Android > OpenXR.
  • Тест на пристрої: Для Android — увімкніть USB Debugging (Settings > Developer Options). Для iOS — Apple Developer Program (безкоштовно для тестів).
Ілюстрація налаштування емуляторів: Для тестування AR/VR на пристроях.

2. Створення та накладення 3D-моделей

Крок 2.1: Встановлення Blender (для моделювання)

  • Посилання: https://www.blender.org/download/
  • Платність: Безкоштовно.
  • Розмір: Завантаження ~200-300 MB, встановлений ~1.5 GB.
  • Час установки: 2-3 хв.

Покрокова установка:

  1. Завантажте останню версію (Blender 4.5+).
  2. Запустіть інсталятор > "Next" > Install.
Ілюстрація Blender: Інструмент для створення 3D-моделей для AR/VR.

Крок 2.2: Створення моделі покроково

  1. Відкрийте Blender. Натисніть "General" шаблон.
  2. Моделювання: View > Add > Mesh > Cube (створіть каркас). Використовуйте інструменти в Toolbar (ліворуч): Extrude (E), Loop Cut (Ctrl+R).
  3. Текстурування: Перейдіть до закладки "Shading" > Add Material > Використовуйте Node Editor для додавання текстур (Image Texture Node > Open image з Photoshop/Substance Painter).
  4. Експорт: File > Export > glTF (.glb) або FBX. Оберіть шлях, натисніть "Export glTF 2.0". (glTF сумісний з Unity AR).
Ілюстрація процесу моделювання: Покрокове створення 3D-об'єктів у Blender.

Крок 2.3: Імпорт у Unity

  1. У Unity: Assets > Import Package > Custom Package (або перетягніть файл .glb/.fbx у Project вікно).
  2. Перетягніть модель на Scene. Додайте компоненти: GameObject > XR > AR Session (для AR), AR Session Origin.
  3. Визначення якоря: Додайте AR Anchor Manager до AR Session Origin. У скрипті (C#): Використовуйте ARRaycastManager для hit-testing (промінь на площину).
  4. Відстеження: AR Foundation використовує SLAM автоматично. У Inspector: AR Plane Manager > Detection Mode = Horizontal/Vertical.
  5. Рендеринг: Додайте AR Camera Manager до Main Camera.
Ілюстрація імпорту: Накладення 3D-моделей у Unity для AR.

3. Додавання інтерактивності

Інтерактивність — реакція на дотики, жести, голос тощо, як ви описали.

Крок 3.1: Базова інтерактивність

  1. Сцена: У Hierarchy створіть ієрархію (Scene Graph): Parent Object > Child Model з Transform.
  2. Hit-testing: Додайте AR Raycast Manager до AR Session Origin. Створіть скрипт (Assets > Create > C# Script): using UnityEngine.XR.ARFoundation; public class PlaceObject : MonoBehaviour { public ARRaycastManager raycastManager; void Update() { if (Input.touchCount > 0) { Touch touch = Input.GetTouch(0); List<ARRaycastHit> hits = new List<ARRaycastHit>(); raycastManager.Raycast(touch.position, hits, TrackableType.Planes); if (hits.Count > 0) { /* Інстанціюйте модель на hits[0].pose */ } } } } Прикріпіть до AR Session Origin.
  3. Жести: Використовуйте MRTK: Додайте MRTK Input System. Для pinch/rotate — Object Manipulator компонент на модель.
  4. Голос: Інтегруйте Speech Recognition (Window > Package Manager > "Windows Speech" для Windows, або third-party як Google Cloud Speech-to-Text API).
    • Приклад: У скрипті використайте KeywordRecognizer для команд ("open door").
  5. Eye-tracking: Для Quest — використовуйте Oculus Integration (Asset Store > "Oculus Integration" > Import). Додайте EyeTracking компонент.
Ілюстрація інтерактивності: Жести, дотики та голосове керування в AR/VR.

Крок 3.2: Анімації

  1. У Blender: Створіть анімацію (Animation закладка > Insert Keyframe).
  2. Експорт з анімацією в glTF.
  3. У Unity: Додайте Animator компонент > Create Animator Controller > Додайте стани (Idle, Open).
  4. У скрипті: animator.SetTrigger("Open") на подію (наприклад, touch).
Ілюстрація анімацій: Динамічні ефекти в 3D-моделях.

4. Тестування та деплоїмент

  1. Тест у Editor: Play Mode з XR Simulation (Window > XR > Simulation > Enable).
  2. Build: File > Build Settings > Add Open Scenes > Platform = Android/iOS > Build & Run (під'єднайте пристрій).
  3. Для Quest: Змініть платформу на Android, додайте OpenXR. Build APK > Перенесіть на Quest через SideQuest (https://sidequestvr.com/, безкоштовно).
  4. Оптимізація: Використовуйте Profiler (Window > Analysis > Profiler) для CPU/GPU.
Ілюстрація тестування: Деплоїмент та оптимізація AR/VR-додатків.

Альтернатива: Unreal Engine

Якщо Unity не підходить (для фотореалізму):

  • Посилання: https://www.unrealengine.com/en-US/download
  • Платність: Безкоштовно.
  • Розмір: ~35 GB завантаження, встановлений ~115 GB (можна зменшити, виключивши шаблони).
  • Час: 1-3 години.
  • Установка: Завантажте Epic Games Launcher > Install UE 5.6+ > Options > Виключіть непотрібні (e.g., no iOS якщо не потрібно).
  • AR/VR: Project > Plugins > Enable AR Utilities, OpenXR.
Ілюстрація Unreal Engine: Альтернатива для фотореалістичних AR/VR-проектів.

Цей посібник дозволяє створити складний додаток, наприклад, AR-меблі з жестами та анімаціями. Якщо потрібно розширити (e.g., multiplayer), додайте Photon або Mirror. Почніть з простої сцени та додавайте складність. Успіхів!
